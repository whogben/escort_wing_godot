# Escort Wing
Top-down 2d flying game and winner of the 2005 Original Mac Games Cup.

## [Click here to play in browser](https://whogben.github.io/escort_wing_godot/)

## Ported to Godot
This Godot port optimizes for nostalgia, replicating the May 2006 community edition's gameplay and graphics as closely as possible.

## Mods
Mods are supported and original mod dirs can be loaded from the main menu.
